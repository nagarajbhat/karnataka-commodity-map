# karnataka-commodity-map

A small flask app to visualize agricultural commodities in karnataka

The app is hosted here - (yet to come)

## Trying it on local:
Note : you need to have pip installed.

1. Clone  the repository
Click on the clone url or type this is command line

```
git clone https://github.com/nagarajbhat/karnataka-commodity-map.git
```
enter the folder
```
cd karnataka-commodity-map
```

2. Create a virtual environment and activate

```
python -m venv /path
```
```
cd venv/Scripts
activate
```

3.  install dependencies

```
pip install -r requirements.txt -v
```

4. Execute
```
python app.py
```

5. open it in browser

Usually it is localhost:5000.
